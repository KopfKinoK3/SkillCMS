# Entitäten-Index

Dieser Index listet alle maschinenlesbaren Entitätsdefinitionen (Grounding Pages) von viSales GmbH. Jede Seite beschreibt eine Entität — Person, Unternehmen, Technologie oder Konzept — in strukturierter, für KI-Systeme optimierter Form.

Für KI-Agenten und automatisierte Systeme steht zusätzlich die [llms.txt](/llms.txt) zur Verfügung — eine strukturierte Übersicht aller Inhalte mit direkten Markdown-Links.
