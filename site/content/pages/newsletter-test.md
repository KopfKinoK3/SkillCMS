# Newsletter Shortcode Test

Dies ist eine Test-Seite für den neuen `[newsletter: "..."]` Shortcode.

Der Shortcode wird direkt im Markdown eingebunden und von `build.py` verarbeitet.

## Beispiel: Newsletter eingebunden

[newsletter: "Fallbeispiele aus 3D & Vertrieb — für Vertriebsleiter und Geschäftsführer"]

Das ist ein Test, dass der Newsletter-Block perfekt zwischen Artikel-Content platziert werden kann.
